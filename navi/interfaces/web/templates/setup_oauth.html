OAuth Setup Required

Setup needed: The Google OAuth credentials file is missing. You need to set up OAuth credentials to use NAVI's web interface.

Follow these steps to set up OAuth:

1

Go to Google Cloud Console

Visit the Google Cloud Console Credentials page

2

Create OAuth 2.0 Client ID

Click "Create Credentials" ‚Üí "OAuth 2.0 Client ID"

Choose "Web application" as the application type

3

Configure Redirect URIs

Add these authorized redirect URIs:

http://localhost:4999/auth/callback
http://127.0.0.1:4999/auth/callback
4

Download credentials.json

Download the JSON file and save it as:

../credentials.json

(One directory up from the src folder)

5

Restart the server

Stop and restart the NAVI web server for the changes to take effect

Important: Make sure the credentials.json file is in the correct location and contains valid OAuth 2.0 credentials. The web application redirect URIs must match exactly.
